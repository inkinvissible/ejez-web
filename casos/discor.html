Discor: Migración de ERP, Catálogo B2B e IA para Búsqueda Visual

Cómo un distribuidor B2B de autopartes en Córdoba migró de un ERP enlatado costoso a un sistema custom con catálogo inteligente y búsqueda visual por foto usando transfer learning.

Cliente Discor
Industria Distribución B2B
Resultado -75% costos operativos

El Desafío

Un ERP "moderno" construido con tecnología del 2000

En 2018, un distribuidor B2B de autopartes en Córdoba contrató un ERP desarrollado por una empresa local de software para cumplir con los nuevos requisitos de facturación electrónica. Lo que prometía ser una solución moderna resultó ser un sistema con tecnología obsoleta: interfaz confusa, caídas constantes, y un costo mensual de USD 200 que no justificaba el servicio.

El sistema fallaba en momentos críticos — durante cierres de mes, emisión de facturas y consultas de stock. Cada caída significaba llamar al soporte, esperar horas y perder ventas. Además, los clientes de Discor no tenían forma de consultar el catálogo de productos de manera autónoma, lo que generaba una carga operativa constante para el equipo comercial.

Antes

Sistema Enlatado

USD 200/mes, Caídas constantes del sistema, Sin catálogo digital para clientes, Búsqueda de productos 100% manual.

Después (EJEZ)

Sistema Custom + IA

USD 50/mes, 99.9% uptime garantizado, Catálogo B2B con login de clientes, Búsqueda visual con IA por foto.

El Reto Técnico

Arqueología de bases de datos e integración de hardware

Este proyecto nos exigió resolver dos desafíos de altísima complejidad técnica y operativa:

El primero fue la migración de datos desde un motor Firebird sumamente antiguo (tecnología de principios de los 2000 que jamás había sido actualizada). El ERP viejo tenía una estructura de base de datos denormalizada de forma caótica; las relaciones lógicas no estaban documentadas y los JOINs eran tan costosos computacionalmente que el software original metía casi todo en una misma tabla gigante. Descifrar ese laberinto para normalizarlo y migrar el historial comercial a un esquema moderno fue una verdadera tarea de arqueología de software.

El segundo fue la integración de hardware con las impresoras de etiquetas Zebra del depósito. En lugar de depender de los inestables drivers genéricos de Windows (que se desconfiguraban constantemente interrumpiendo el despacho), programamos la comunicación directa con los equipos. Diseñamos la integración escribiendo directamente en el lenguaje nativo de la impresora (ZPL - Zebra Programming Language), garantizando una impresión de códigos de barras instantánea, precisa y 100% robusta.

La Solución

Sistema custom, catálogo B2B inteligente y búsqueda visual con IA

La solución fue integral y se construyó en tres etapas:

Primero, migramos toda la operación a un ERP custom desplegado en infraestructura self-hosted. El nuevo sistema maneja facturación electrónica, gestión de stock, y reportes — todo por USD 50/mes con 99.9% de disponibilidad.

Segundo, desarrollamos un catálogo digital B2B con dos niveles de acceso: un catálogo público para que cualquier visitante explore los productos, y un portal privado con login donde los clientes registrados ven precios, disponibilidad y pueden generar pedidos.

Tercero, y esto es lo más innovador: implementamos un sistema de búsqueda visual de autopartes usando transfer learning. Entrenamos un modelo de deep learning con las imágenes del catálogo de Discor para que los clientes puedan sacar una foto a la pieza que necesitan y el sistema identifique automáticamente el producto correspondiente en el catálogo. Esto reduce drásticamente el tiempo de búsqueda y elimina errores de identificación.

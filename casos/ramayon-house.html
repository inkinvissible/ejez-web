Ramayón House: Sistema de Precios Dinámicos y Web Multi-Page

Cómo transformamos la presencia digital de un apart-hotel en Carlos Paz con un sitio multi-page a medida y un sistema de precios dinámicos que automatiza la gestión tarifaria por temporada.

Cliente Ramayón House
Industria Alojamientos Turísticos
Ubicación San Martín de los Andes
Tipo Multi-page + Sistema a medida

El Desafío

Gestión manual de tarifas y presencia digital limitada

Ramayón House es un conjunto de departamentos situado en San Martín de los Andes que necesitaba dar el salto digital. Su principal desafío era doble: por un lado, no contaban con un sitio web que transmitiera la calidad real de sus instalaciones. Por otro, la gestión de tarifas por temporada era completamente manual — actualizaban precios en distintas plataformas una por una, con errores frecuentes y horas invertidas en cada cambio de temporada.

  • Sin sitio web propio — dependían exclusivamente de plataformas de terceros (Booking, Airbnb)
  • Gestión de tarifas manual con planillas de Excel por cada unidad y temporada
  • Inconsistencia de precios entre canales
  • Imposibilidad de mostrar todas las cabañas, servicios y entorno de manera atractiva

El Reto Técnico

Seguridad en la integración de APIs y funciones serverless

El principal obstáculo en este desarrollo no estuvo en diseñar la interfaz visual del calendario, sino en la arquitectura de comunicación segura con la API de Airbnb para consultar y sincronizar la disponibilidad en tiempo real.

Realizar peticiones HTTP directas desde el propio navegador del usuario (frontend) hacia la API externa planteaba dos problemas críticos: por un lado, exponía públicamente en el código cliente las credenciales y llaves secretas de autenticación del apart-hotel; por el otro, chocaba contra las políticas de CORS (Cross-Origin Resource Sharing) del navegador, bloqueando la conexión de forma sistemática.

Cómo lo resolvimos: Diseñamos una arquitectura desacoplada y segura. Desarrollamos y desplegamos una función serverless (serverless function) en el backend que actúa como un proxy seguro. Cuando un visitante consulta el calendario, la web se comunica con esta función intermediaria; esta inyecta las credenciales encriptadas en el lado del servidor, realiza la consulta de manera segura a Airbnb y devuelve al frontend la información procesada y limpia. Logramos una sincronización en tiempo real robusta y veloz que protege al 100% las llaves de acceso del negocio.

La Solución

Sitio web multi-page con sistema de precios dinámicos integrado

Diseñamos y desarrollamos un sitio web multi-page completo que captura la esencia del lugar, combinado con un sistema de precios dinámicos construido a medida.

El sitio presenta cada unidad de alojamiento con galería fotográfica, características detalladas y disponibilidad. Además muestra disponibilidad con calendario interactivo conectado por la API de Airbnb. Pero lo verdaderamente diferencial es el cuadro tarifario interactivo: un sistema que permite a Ramayón gestionar precios por temporada, tipo de unidad y cantidad de noches desde un solo lugar, con actualización inmediata en el sitio.
